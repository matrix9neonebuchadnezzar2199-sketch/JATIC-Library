# ユーザーマニュアル（ドラフト）

P10 完了時に本番化します。現時点では [README.md](../README.md) の「このツールでできること」を参照してください。

## 初回起動

1. 保存先フォルダを指定
2. ダウンロード対象地域を選択（既定: 全51地域）
3. 起動時自動チェックの ON/OFF を選択

## 主要操作

| 操作 | 場所 |
|------|------|
| 今すぐ更新確認 | 設定タブ / メニュー「ツール」 |
| サイト再スキャン | メニュー「ツール」 |
| 保管庫を開く | 保管庫タブ |
| テーマ切替 | 設定タブ / メニュー「表示」 |
