# ユーザーマニュアル（ドラフト）

**配布 exe 確定後に本番化します。** 未完成項目は [DEV_STATUS.md §5 残作業](DEV_STATUS.md#5-残作業リリース配布まで)（R5）を参照。

## 初回起動

1. 保存先フォルダを指定（設定タブ）
2. ダウンロード対象地域を選択（既定: 全 51 地域）
3. 「PC起動と同時にチェック」: 起動時自動チェック ON ＋「Windows 起動時に自動起動」ON → 設定を保存

## 主要操作

| 操作 | 場所 |
|------|------|
| 今すぐ更新確認 | 設定タブ / メニュー「ツール」 / トレイ |
| サイト再スキャン | メニュー「ツール」 |
| 保管庫を開く | 保管庫タブ |
| 再 DL・削除・タグ | 保管庫ツリー右クリック（ファイル） |
| 月次エクスポート | 保管庫ツリー右クリック（月ノード） |
| テーマ切替 | 設定タブ / メニュー「表示」 |
| Git 自動 commit | 設定タブ（リポジトリパス・有効化） |

## 配布版（予定）

- 起動: `JATIC-Library.exe`（同フォルダに DLL 等を同梱）
- 初回のみ Playwright 用 Chromium のセットアップが必要な場合あり（手順は R2 確定後に追記）
