# JATIC-Library ユーザーマニュアル

**対象バージョン:** v0.1.0-beta.1（Windows 10 / 11）

配布 zip を展開したフォルダから `JATIC-Library.exe` を起動してください。exe 単体のコピーでは動作しません。

---

## 1. 初回セットアップ

1. zip を任意のフォルダに展開（例: `C:\Apps\JATIC-Library\`）
2. `JATIC-Library.exe` を起動
3. **設定**タブで **保存先フォルダ** を指定（例: `C:\Apps\JATIC-Library\data`）
4. 必要に応じて **対象地域** を選択（未選択＝全 51 地域）
5. **「設定を保存」** を押す（タイトルバーの `*` が消えるまで未保存）
6. 初回の **「サイト再スキャン」** または **「今すぐ更新確認」** で、Chromium セットアップダイアログが出たら **「はい」** で約 200MB をダウンロード（インターネット必須）

詳細は [BETA_TEST.md](BETA_TEST.md) を参照。

---

## 2. 画面構成

| タブ | 用途 |
|------|------|
| **保管庫** | 取得済み ZIP の閲覧・再 DL・削除・エクスポート |
| **設定** | 保存先・地域・スケジュール・テーマ・Git 連携 |

メニュー **ファイル / ツール / 表示 / ヘルプ** とシステムトレイからも同じ操作にアクセスできます。

---

## 3. 設定タブ

| 操作 | 説明 |
|------|------|
| 保存先の参照… | DL 先ルート（`YYYY_M` フォルダがその下に作られます） |
| 地域チェック | 空＝全地域。変更後は **設定を保存** が必要 |
| PC起動と同時にチェック | ON でログオン後に自動起動＋24h 以内なら更新確認 |
| Windows 起動時に自動起動 | レジストリ Run キーに登録 |
| 今すぐ更新確認 | 手動でチェック＋必要なら DL（進捗ダイアログ） |
| サイト再スキャン | JARTIC から typeB リンクを学習し `targets.json` 更新 |
| テーマ | ライト / ダーク（保存で反映） |
| Git 自動 commit | リポジトリパスと有効化（commit のみ、push は手動） |

未保存の変更があるとき、ウィンドウタイトルに `*` が付き、**更新確認・再スキャンはブロック**されます。先に保存してください。

---

## 4. 保管庫タブ

| 操作 | 説明 |
|------|------|
| 再読込 | ディスクを再スキャンしてツリー更新 |
| ソート | 並び順を変更（300ms デバウンスで設定保存） |
| 検索 | 地域名で表示フィルタ（永続化なし） |
| ファイル選択 | 右ペインにサイズ・SHA256・タグ等 |
| 右クリック（ファイル） | エクスプローラーで開く / パスコピー / 再 DL / 削除 / タグ管理 |
| 右クリック（月ノード） | 月次 ZIP バンドル or 統合 CSV エクスポート |

**削除**は確認ダイアログのあと、DB → manifest → ファイルの順で削除します。ファイル削除に失敗した場合は警告が表示されます。

---

## 5. メニュー・トレイ

| 操作 | 場所 |
|------|------|
| 今すぐ更新確認 | ツールメニュー / 設定タブ / トレイ |
| サイト再スキャン | ツールメニュー / 設定タブ |
| ライト / ダークテーマ | 表示メニュー / 設定タブ |
| バージョン情報 | ヘルプメニュー |
| 終了 | ファイルメニュー / トレイ「終了」 |

トレイが有効なとき、ウィンドウの **×** は **終了ではなく非表示** です。完全終了はトレイの **終了** またはメニュー **ファイル → 終了** を使ってください（SQLite WAL の checkpoint が走ります）。

---

## 6. データの場所

| 種類 | パス |
|------|------|
| 設定・DB・ログ | `%LOCALAPPDATA%\JATIC-Library\` |
| ダウンロード ZIP | 設定した保存先の `YYYY_M\` 以下 |

## 7. トラブルシューティング

| 症状 | 対処 |
|------|------|
| SmartScreen / Defender の警告 | 「詳細情報」→「実行」。未署名 exe のため表示され得ます |
| 更新確認が始まらない | 設定タブで保存先を設定し **設定を保存**。未保存 `*` がないか確認 |
| 再スキャン・更新確認で止まる | Chromium 未導入 → セットアップダイアログでインストール |
| DL が 404 | 自動で 1 回再スクレイプ後に再試行（それでも失敗はログ確認） |
| HTTP/2 エラー | 自動で HTTP/1.1 にフォールバック |
| CSV が文字化け（Excel） | **地域 ZIP 内の CSV** は JARTIC 配布のまま（バイナリ保存）。**統合.csv** および「統合 CSV エクスポート」は **Shift_JIS（cp932）** で出力する（Excel のダブルクリック向け）。旧版の UTF-8 出力では、見出し行だけ日本語が化けてデータ行は数字中心のため正常に見えることがある（Excel が UTF-8 BOM なしを CP932 と誤判定するため）。地域 ZIP だけが化ける場合は、同様に開き方・文字コードを確認する |
| ログ | `%LOCALAPPDATA%\JATIC-Library\logs\`（GUI exe はファイルのみ） |
| 設定をリセット | `%LOCALAPPDATA%\JATIC-Library\` を退避または削除（再起動で初期化） |

## 8. 既知の制限

- Windows 10 / 11 のみ
- 過去月の遡及 DL は不可（JARTIC の公開範囲外）
- カレンダー・比較タブは未実装（将来バージョン）
- Git 自動 **push** は未対応
